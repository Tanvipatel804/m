Phase 02 Add Your First Driver
✓
Navigate to Fleet → Drivers → Add New Driver
Enter full legal name, date of birth, home terminal location, and employment type (company driver vs. owner-operator).
Required
✓
Enter CDL number, class, and endorsements
Class A/B/C, plus endorsements (H, N, T, X, etc.). The TMS cross-checks endorsements against load requirements automatically.
Required
✓
Upload CDL front and back
Accepted formats: JPG, PNG, PDF. The system will parse expiry date and alert you 60 days before renewal.
Document
✓
Upload Medical Examiner's Certificate (MEC)
Required for interstate commerce. Must be current (valid date within last 24 months). FMCSA National Registry number is auto-verified.
Document
✓
Confirm drug & alcohol testing enrollment
Indicate the driver's testing consortium/C&TPA. If enrolling through the TMS partner program, click "Enroll Now" to initiate a pre-employment test order.
Required
✓
Send driver a mobile app invite
Driver receives an SMS/email link to download the Driver App. They set a PIN to activate ELD and messaging features.

Phase 03 Add Your First Truck
✓
Navigate to Fleet → Assets → Add New Vehicle
Select vehicle type (Dry Van, Reefer, Flatbed, Tanker, etc.) and enter VIN, make, model, and year.
Required
✓
Enter USDOT number and license plate details
Plate number, state of registration, and expiry date. Add trailer unit number if applicable. The TMS validates the USDOT against FMCSA records.
Required
✓
Upload vehicle registration and IFTA credentials
Current cab card (registration), plus IFTA license for interstate fuel tax reporting. The TMS uses this for auto-generated fuel tax summaries.
Document
✓
Link ELD device to the truck record
Enter the ELD provider and device serial number. The TMS supports Samsara, KeepTruckin/Motive, Omnitracs, and others via API. HOS data syncs automatically once paired.
Required
✓
Set payload capacity and temperature range (if reefer)
Max gross weight, usable payload (lbs), and interior dimensions. Reefer units: enter min/max temperature capability. Used for automatic load matching.
Setup Tip
✓
Schedule or confirm last annual inspection (DOT)
Upload the most recent annual inspection report. The TMS will flag loads where inspection is expired or within 30 days of expiry.
Document
Phase 04 Assign & Verify
✓
Assign the driver to the truck
In Fleet → Driver record, use the "Assign Asset" dropdown to link the truck. This creates the Driver–Truck pairing used for dispatch and HOS tracking.
Required
✓
Run a compliance readiness check
Go to Compliance → Readiness Report and run a check on the new driver–truck pair. Resolve any red-flag items (expired docs, missing fields) before accepting a load.
Required
✓
Confirm ELD telemetry is live
In the Dispatch map view, verify the truck shows a live GPS ping. If offline, follow the ELD troubleshooting guide under Help → ELD Pairing.
Setup Tip
✓
Accept your first load or post available capacity
Your driver–truck pair is now eligible for dispatch. Browse the Load Board or accept a tendered load. The system will auto-populate driver and truck details on the BOL.
